# Security

Please report vulnerabilities privately through GitHub Security Advisories when available. Do not include session cookies, paid article content, or other credentials in public issues.

Offprint processes untrusted web content and can launch Chromium. Keep `--disable-web-security` disabled unless the input is trusted. Stored cookies live outside the repository with user-only permissions, but plaintext cookie files should still be protected.

Only the latest release receives security fixes.
